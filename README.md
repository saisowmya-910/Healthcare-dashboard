# Healthcare Dashboard

## Overview
Developed an interactive Power BI Healthcare Dashboard to analyze patient records, hospital operations, and treatment performance. The dashboard provides insights into patient demographics, admissions, diagnoses, and healthcare trends to support data-driven decision-making.

## Tools Used
- Power BI
- Excel
- Data Analysis
- Data Visualization

## Key Features
- Patient Demographics Analysis
- Hospital Admission and Discharge Trends
- Department-wise Performance Tracking
- Disease and Diagnosis Analysis
- Treatment Outcome Monitoring
- Interactive Filters and Drill-Down Analysis
- KPI Monitoring Dashboard

## Business Insights
- Identifies trends in patient admissions and discharges.
- Monitors department performance and resource utilization.
- Analyzes common diseases and treatment patterns.
- Supports healthcare decision-making through visual analytics.
- Improves operational efficiency using data-driven insights.

## Skills Demonstrated
- Data Cleaning and Transformation
- Data Modeling
- DAX Calculations
- Dashboard Design
- Business Intelligence Reporting
- Data Visualization
- Analytical Thinking
